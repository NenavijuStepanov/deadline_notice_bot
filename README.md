# Deadline-bot
<b>@deadline_notice_bot</b>

ТГ бот для взаимодействия с напоминаниями о дедлайнах

# Команды:

<code>/start</code> - подсказки

<code>/add Название / ЧЧ:ММ / ГГГГ-ММ-ДД / Описание</code> - добавление дедлайнов

<code>/list</code> - список активных дедлайнов(+ кнопки удаления)

<code>/archive</code> - список истекших дедлайнов

# Параметры

Изменение названия БД и токена бота в .env
